# FlightSearch - 航空券予約システム

## 🎉 予約システムフロント画面モック完成！

**完全な予約フローが実装されました！**

### 予約フロー（全5ステップ）
1. **フライト検索** → 出発地、目的地、日付、乗客数を入力して検索
2. **フライト選択** → 検索結果から希望のフライトを選択
3. **座席クラス選択** → 往路・復路それぞれの座席クラスを選択
4. **座席選択・お客様情報入力** → シートマップで座席を選択し、乗客情報を入力
5. **予約内容の最終確認** → すべての情報を確認
6. **決済（Stripe風）** → クレジットカード情報を入力
7. **予約完了** → 予約番号発行、確認書ダウンロード、トップへ戻る

## 航空会社ロゴ実装

### ロゴの取得方法
- **ソース**: Google Flights CDN (gstatic.com)
- **URL形式**: `https://www.gstatic.com/flights/airline_logos/70px/{IATA_CODE}.png`
- **対応航空会社**:
  - ANA (NH): 全日空
  - JAL (JL): 日本航空
  - UA: ユナイテッド航空
  - DL: デルタ航空
  - AA: アメリカン航空

### Amadeus API統合後の動作
- Amadeus APIから取得した航空会社コード（IATA）を使用
- 上記URL形式で航空会社ロゴを自動表示
- フォールバック: ロゴ読み込み失敗時は航空会社コードを表示

## 使用方法

### 完全な予約フロー（7ステップ）

#### 1. フライト検索
- 出発地、目的地を入力（例: 東京 → ニューヨーク）
- 出発日、帰国日を選択
- 乗客数を設定（大人、子供、幼児）
- 座席クラスを選択
- 「フライトを検索」ボタンをクリック

#### 2. 検索結果の確認
- 検索結果から希望のフライトカードを表示
- フィルター・ソート機能で絞り込み
- 「選択する」ボタンをクリック

#### 3. 座席クラス選択（モーダル）
- 往路便の座席クラスを選択（エコノミー、プレミアムエコノミー、ビジネス、ファースト）
- 復路便の座席クラスを選択
- 合計金額をリアルタイム確認
- 「予約を続ける」ボタンをクリック

#### 4. 座席選択・お客様情報入力
- **シートマップ**から希望の座席を選択（人数分×往復）
- **乗客情報**を入力（全員分）
  - 姓、名、性別、生年月日
  - パスポート番号、国籍
- **連絡先情報**を入力
  - メールアドレス、電話番号
  - 緊急連絡先（任意）
- 利用規約に同意
- 「予約内容の確認」ボタンをクリック

#### 5. 予約内容の最終確認
- **フライト情報**の確認（便名、区間、座席クラス、選択座席）
- **乗客情報**の確認（全員分の詳細）
- **連絡先情報**の確認
- **お支払い合計金額**の確認
- 「支払いへ進む」ボタンをクリック

#### 6. 決済（Stripe風）
- **セキュリティ表示**の確認（SSL/TLS暗号化）
- **ご注文内容**の確認（料金内訳）
- **お支払い情報**の入力
  - カード番号
  - 有効期限（MM/YY）
  - セキュリティコード（CVV）
  - カード名義人
  - 請求先住所
- 「¥XXX を支払う」ボタンをクリック

#### 7. 予約完了
- **予約番号**が発行される
- **フライト情報サマリー**を確認
- **次のステップ**を確認
  1. 予約確認メールを確認
  2. オンラインチェックイン（出発24時間前から）
  3. 空港へ出発（出発2時間前まで）
- **アクション**
  - 「予約確認書をダウンロード」（ダミー実装）
  - 「検索トップに戻る」でフロー完了
